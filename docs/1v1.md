## 规则引擎产生的游戏后端服务器事件

### decide_next_action 产生后续服务器需要处理的事件
- {:table, {:notify_blind_bet, %{0 => 5, 1 => 10}}}
- {:table, {:deal, :flop}}
- {:table, {:deal, :turn}}
- {:table, {:deal, :river}}
- {:player, {1, [:fold, {:call, 5}, :raise]}}
- {:winner, 1, %{0 => 90, 1 => 110}}
- {:show_hands, 170, %{0 => 415, 1 => 415}}

### handle_action 根据事件演进规则引擎
- {:table, :notify_blind_bet_done}
- {:table, {:done, :flop}}
- {:table, {:done, :turn}}
- {:table, {:done, :river}}
- {:player, :fold}
- {:player, :call}
- {:player, {:raise, x}}
